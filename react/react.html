React

Programas necesarios

  1. Node.js para el desarrollo local
  2. Next.js es un framework de React. Se ejecuta con el comando:
    npx create-next-app@latest

Comandos iniciales en consola

  1. cd RUTA-APP
  2. npm start inicia el servidor de desarrollo en http://localhost:3000/
  3. npx create-react-app CARPETA crea una app en la carpeta señalada
  4. npx create-react-app . crea una app en la carpeta actual
  5. npm run build agrupa la aplicación en archivos estáticos para producción
  6. npm test inicia el corredor de pruebas
  7. npm run eject Elimina esta herramienta y copia las dependencias de compilación, los archivos de configuración y los scripts en el directorio de la aplicación. Esta acción no puede ser revertida
  8. Ctrl+C para cerrar el servidor de desarrollo

Exportar un componente

  1. Exportanción por defecto. Se define al final del archivo del componente que se quiera exportar

    export default NOMBRECOMPONENTE

  2. Exportanción nombrada. Exporta un determinado componente de un archivo de componentes. Se define al comienzo de la funcion del elemento a exportar

    export function Article() { return ( ... ); }

Importar un componente

  1. Importando un componente exportado por defecto. Se define en la parte superior del documento que importará el componente

    import NOMBRECOMPONENTE from './CARPETA/ARCHIVO.js';

  2. Importando un componente nombrado. Se define en la parte superior del documento que importará el componente

    import {NOMBRECOMPONENTE} from './CARPETA/ARCHIVO.js';

Estructura básica de un componente

  1. Importando los elementos del componente

    import React from "react"; //Importa los componentes de React import '../css/imagen.css'; //Importa la hoja de estilo de este componente import paisaje from '../img/paisaje1.jpg'; //Importa la imagen usada en el componente

  2. Se crea una función que retorne el elemento a renderizar en App.js

    //Definiendo las propiedades con la sintaxis de desestructuración function Imagen ({altImagen, tituloImagen}) { return ( <picture className="imagen-paisaje"> <img src={paisaje} alt={altImagen} title={tituloImagen} /> </picture> ) }
    //-------------------------------------- //Definiendo las propiedades con la sintaxis normal function Imagen (props) { return ( <picture className="imagen-paisaje"> <img src={paisaje} alt={props.altImagen} title={props.tituloImagen} /> </picture> ) } //Exporta el componente para su posterior importación en la App principal export default Imagen;
