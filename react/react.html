NPM (Node Package Manager)

Es un gestor de paquetes para los proyectos de Node.js que se encuentran disponibles para su uso público. Estos proyectos están en el registro de npm y se denominan paquetes.

  1. Comprobar versión instalada

    npm -v
    npm --version

  2. Instalar/actualizar la última versión

    npm install -g npm@latest

Comandos más comunes de la consola

  1. cd RUTA para moverse entre directorios
  2. npm start inicia el servidor de desarrollo en http://localhost:3000/
  3. npx create-react-app carpeta-app crea una App en la carpeta señalada. No se aceptan mayúsculas para el nombre
  4. npx create-react-app . crea una App en la carpeta actual
  5. npm install LIBRERIA instala una librería para el uso de sus componentes pre fabricados
  6. npm uninstall LIBRERIA desinstala la librería indicada
  7. npm run build agrupa la aplicación en archivos estáticos para producción
  8. npm test inicia el corredor de pruebas
  9. npm run eject Elimina esta herramienta y copia las dependencias de compilación, los archivos de configuración y los scripts en el directorio de la aplicación. Esta acción no puede ser revertida
  10. Ctrl+C para cerrar el servidor de desarrollo

Estructura básica de un componente

Son funciones que se definen en archivos JSX independientes y que luego se llaman desde la App principal donde serán renderizados. Los nombres dentro del código siempre se definen con la primera letra en mayúscula

  1. Importando los elementos del componente

    import React from "react"; //Importa los componentes de React import '../css/imagen.css'; //Importa la hoja de estilo de este componente import paisaje from '../img/paisaje1.jpg'; //Importa la imagen usada en el componente

  2. Se crea una función que retorne el elemento o estructura a renderizar en App.jsx

    //Definiendo las propiedades con la sintaxis normal function Imagen (props) { return ( <picture className="imagen-paisaje"> <img src={paisaje} alt={props.altImagen} title={props.tituloImagen} /> </picture> ) } //Exporta el componente para su posterior importación en la App principal export default Imagen;
    //Definiendo las propiedades con la sintaxis de desestructuración function Imagen ({altImagen, tituloImagen}) { return ( <picture className="imagen-paisaje"> <img src={paisaje} alt={altImagen} title={tituloImagen} /> </picture> ) } //Exporta el componente para su posterior importación en la App principal export default Imagen;

  3. Definiendo el componente con una función flecha

    const Imagen = ({altImagen, tituloImagen}) => { return ( <picture className="imagen-paisaje"> <img src={paisaje} alt={altImagen} title={tituloImagen} /> </picture> ) }

Exportar un componente

  1. Exportanción por defecto. Se define al final del archivo del componente que se quiera exportar

    export default NombreComponente;

  2. Exportanción nombrada. Exporta un determinado componente de un archivo de componentes. Se define al comienzo de la funcion del elemento a exportar

    export function Article() { return ( ... ); }

Importar un componente

  1. Importando componentes exportados por defecto. Se definen en la parte superior del documento que los importará

    import Componente1 from './carpeta/Archivo1.jsx'; import Componente2 from './carpeta/Archivo2.jsx';

  2. Importando componentes nombrados. También se definen en la parte superior del documento que los importará

    import {Componente1, Componente2} from './carpeta/Archivo.jsx'; import {Componente3} from './carpeta/Archivo3.jsx';

  3. Importando ambas clases de componentes desde un mismo archivo

    import Componente1, {Componente2} from './carpeta/Archivo.jsx';

Hooks

Los hooks son funciones que agregan ciertas capacidades a la aplicación. Los más comunes son:

  • useState
  • useEffect
  • useState

Importar un hook

import {useState} from 'react';
import {useState, useEffect} from 'react';

Hook useState

Es un arreglo compuesto por una variable y una función que modifica esa variable, por lo tanto se debe ejecutar en la aplicación asignándole ambos elementos

import React, { useState } from 'react'; const HookUseState = () => { const [value, setValue] = useState(0) const updateValue = () => { setValue(value + 1) } return ( <button onClick={updateValue}> {value} </button> ) } export default HookUseState

props

Son variables cuyo valores se pasan desde la App principal hacia los componentes que renderiza

  1. Se llaman en el componente como parámetro de la función y se retornan junto con el nombre asignado en App

    const PropsComponent = (props) => { return ( <h1 className={props.styleColor} title={props.title}>Texto de prueba con {props.title}</h1> ) }
    const PropsComponent = ({styleColor, titles ,title}) => { return ( <h1 className={styleColor} title={title}>Texto de prueba con {title}</h1> ) }

  2. Se definen en la App como variables cuyos valores se llaman desde el componente

    <PropsComponent titles="titulo" title="ahora si" styleColor="propsColor" />

props predefinidas

Son atributos predefinidos en el componente para mostrarlos en caso de que no se hayan definido en la App principal. De este modo, no dará error

const UserData = ({userName = "User name", userLastName = "User last name", age}) => { <ul> <li>{userName}</li> <li>{userLastName}</li> <li>{age}</li> </ul> }

Bucle map

Es un método que permite iterar sobre un array y transformar cada uno de sus elementos en un nuevo valor según una función proporcionada

  • Ejemplo de array en JavaScript

    const taskList = [ { id: 1, title: 'Primera tarea', author: 'Leandro', startDate: '11/04/2024', endDate: '1204/2024' }, { id: 2, title: 'Segunda tarea', author: 'Federico', startDate: '09/04/2024', endDate: '11/04/2024' }, { id: 3, title: 'Tercera tarea', author: 'Ludueña', startDate: '05/04/2024', endDate: '15/05/2024' }, { id: 4, title: 'Programar', author: 'Leandro', startDate: '10/04/2024', endDate: '12/05/2024' }, { id: 5, title: 'Estudiar y testear', author: 'Federico', startDate: '20/04/2024', endDate: '22/05/2024' } ] export default taskList;

  • Llamando al array con .map(), asignándole una variable cualquiera para cada objeto (task) y otra para sus respectivos índices

    import React from 'react'; import taskList from '../taskList.js'; const BucleMap = () => { return ( <div> {taskList.map((task, i) => ( <ul key={i}> <li>{task.title}</li> <li>{task.author}</li> <li>{task.startDate}</li> <li>{task.endDate}</li> </ul> ))} </div> ) } export default BucleMap;

  • Llamando al array nuevamente pero asignando una key a partir del ID de cada objeto

    import React from 'react'; import taskList from '../taskList.js'; const BucleMap = () => { return ( <div> {taskList.map((task) => ( <ul key={task.id}> <li>{task.title}</li> <li>{task.author}</li> <li>{task.startDate}</li> <li>{task.endDate}</li> </ul> ))} </div> ) } export default BucleMap;
