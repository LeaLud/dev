React. Comenzando

Programas necesarios

  • Node.js para el desarrollo local
  • Next.js es un framework de React. se ejecuta con npx create-next-app@latest

Comandos iniciales en consola

  • npx create-react-app CARPETA crea una app en la carpeta señalada
  • npm start inicia el servidor de desarrollo en http://localhost:3000/. Se debe ejecutar en la carpeta de la app
  • npm run build agrupa la aplicación en archivos estáticos para producción
  • npm test inicia el corredor de pruebas
  • npm run eject Elimina esta herramienta y copia las dependencias de compilación, los archivos de configuración y los scripts en el directorio de la aplicación. Esta acción no puede ser revertida

Exportar e importar componentes

  • export default NOMBRECOMPONENTE;. Se agrega al final del archivo del componente que se quiera exportar
  • import './ARCHIVO.EXTENSION'; o import COMPONENTENTE from './ARCHIVO.EXTENSION'; para incluir en un nuevo archivo

Varios

  • npm es el comando que instala paquetes y npx los ejecuta desde una descarga temporal
