JavaScript

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

No requiere ningún tipo de instalación, ya que la mayoría de los navegadores web lo incorporan.

Funciones flecha

  1. Son funciones abreviadas y anónimas (sin nombre) que se utilizan en variables

  2. La palabra clave function se elimina

  3. Las llaves se eliminan cuando la función tiene una sola línea

  4. La palabra clave return se reemplaza por =>

  5. Se escribe toda la función en una sola línea

    function myFunction(b) { return 10 + b } console.log(myFunction(50))
    let myVar = (b) => 10 + b console.log(myVar(50))

  6. Creando un objeto con una función flecha

    let crearUsuario = (Nombre, Oficio, Edad) => ({Nombre, Oficio, Edad}) console.log(crearUsuario("Leandro", "Webmaster", 37))

Condicional ternario

Escribiendo una sentencia condicional normal y con operadores ternarios

a = 9 b = 13 if (a > b) { console.log(a + " es mayor") } else if (a == b) { console.log("Los valores son iguales") } else { console.log(a + " es menor") }

En la condición con operadores ternarios es importante escribir la expresión completa: condición + lo que ocurre si se cumple la condición + lo que ocurre si no se cumple la condición

a = 9 b = 13 console.log(a > b ? a + " es mayor" : a == b ? "Los valores son iguales" : a + " es menor")

Getters y setters

  1. Son métodos para obtener y modificar indirectamente datos de propiedades de objetos que no deberían ser modificadas
  2. Estas propiedades, por convención, se señalan con un guión bajo "_" delante del nombre
  3. Get extrae los datos de una propiedad
  4. Set crea una nueva propiedad con los datos extraídos
class Mascota { constructor(nombre, edad) { this._nombre = nombre this._edad = edad } get nombre() { console.log("Getter") return this._nombre } set nombre(nuevoNombre) { console.log("Setter") this._nombre = nuevoNombre } } const Mascota3 = new Mascota("Federico", 40) console.log(Mascota3.nombre) Mascota3.nombre = "India" console.log(Mascota3.nombre)
