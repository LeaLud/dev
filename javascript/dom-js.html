DOM

Es un modelo de objetos del documento (Document Object Model). Representa la estructura jerárquica de los distintos elementos que componen un documento HTML.

Nodo

Es un punto específico del diagrama o árbol de nodos del DOM. Se reconocen como nodos:

  1. El documento <html></html> (nodo raíz)

  2. Elementos HTML (etiquetas)

  3. Textos

  4. Comentarios

Accediendo a elementos del DOM

  1. Accediendo por ID del elemento

    const varID = document.getElementById("lista") console.log(varID) console.log(varID.innerHTML) console.log(varID.innerText) console.log(varID.textcontent) console.log(varID.tagName)

  2. Accediendo por clase del elemento

    const varClass = document.getElementsByClassName("elemento-lista") console.log(varClass) console.log(varClass[0]) console.log(varClass[0].id) console.log(varClass[0].className)

  3. Accediendo por elemento HTML

    const listTags = document.getElementsByTagName("ul") console.log(listTags)

  4. Accediendo por selector CSS

    const varSelector = document.querySelector(".code-block") console.log(varSelector)
    const varSelector = document.querySelector("#return-to-top") console.log(varSelector)
    const varSelector = document.querySelector("li p") console.log(varSelector)
    const varSelector = document.querySelector("li p span:not(.highlight-text)") console.log(varSelector)
    const varSelector = document.querySelectorAll("li p span:not(.highlight-text)") console.log(varSelector)

Asignando estilos

  1. Accediendo a los estilos del elemento

    console.log(document.querySelector(".boton").style)

  2. Asignando estilos al elemento

    document.querySelector(".boton").style.fontSize = "16px" document.querySelector(".boton").style.padding = "10px" document.querySelector(".boton").style.color = "red"

Manipulando los atributos de un elemento

  1. Accediendo al atributo

    console.log(document.getElementsByTagName("a")[0].getAttribute("href"))

  2. Quitando un atributo

    document.getElementsByTagName("a")[0].removeAttribute("href")

    Asignando / reasignando un valor al atributo

    document.getElementsByTagName("a")[0].setAttribute("href", "https://www.youtube.com/")

Manipulando clases

  1. Accediendo a las clases de un elemento

    console.log(document.querySelector(".ejemplo").classList)
    console.log(document.getElementById("boton-ejemplo").classList)

  2. Agregando clases. Deben estar definidas en <style></style>

    document.querySelector(".ejemplo").classList.add("subrayado", "italic")

  3. Verificando la existencia de una clase

    console.log(document.querySelector(".ejemplo").classList.contains("subrayado"))
    console.log(document.querySelector(".ejemplo").classList.contains("ejemplo"))

  4. Removiendo una clase

    document.querySelector(".ejemplo").classList.remove("ejemplo", "subrayado")

Creando elementos

  1. Agregando varios nodos hijos con la función append()

    let nuevoElemento = document.createElement("li") let nuevoElemento2 = document.createElement("li") nuevoElemento.innerText = "Línea 6" nuevoElemento2.innerHTML = "<strong>Línea 7</strong>" document.getElementById("lista-ejemplo").append(nuevoElemento, nuevoElemento2)

  2. Agregando un nodo hijo a la vez con la función appendChild()

    let nuevoElemento = document.createElement("li") nuevoElemento.innerText = "Línea 6" document.getElementById("lista-ejemplo").appendChild(nuevoElemento)

Removiendo un elemento

document.getElementById("boton-ejemplo").remove()

Manipulando HTML

  1. Accediendo al HTML

    console.log(document.getElementById("lista-ejemplo").innerHTML)

  2. Asignando / reasignando contenido HTML

    document.getElementById("lista-ejemplo").innerHTML = "<ol><li>un item</li><li>dos items</li></ol>"

Manipulando texto

  1. Accediendo al texto

    console.log(document.getElementById("boton-ejemplo").innerText)

  2. Asignando / reasignando texto

    document.getElementById("boton-ejemplo").innerText = "Nuevo botón"

Recorriendo el DOM

  1. Recorriendo elementos y nodos padres

    console.log(document.getElementById("lista-ejemplo").parentElement) console.log(document.getElementById("lista-ejemplo").parentNode) console.log(document.getElementById("lista-ejemplo").parentNode.parentElement)

  2. Recorriendo elementos y nodos hijos

    console.log(document.getElementById("lista-ejemplo").children) console.log(document.getElementById('lista-ejemplo').lastElementChild) console.log(document.getElementById('lista-ejemplo').firstChild)

  3. Recorriendo elementos y nodos hermanos

    console.log(document.getElementById("parrafo-ejemplo").previusElementSibling) console.log(document.getElementById("boton-ejemplo").previusSibling) console.log(document.getElementById("parrafo-ejemplo").nextElementSibling) console.log(document.getElementById("boton-ejemplo").nextSibling) console.log(document.getElementById('parrafo-ejemplo').lastElementChild) console.log(document.getElementById('boton-ejemplo').firstChild)
