DOM

Es un modelo de objetos del documento (Document Object Model). Representa la estructura jerárquica de los distintos elementos que componen un documento HTML.

  • Nodo
  • Accediendo a elementos del DOM

Nodo

Es un punto específico del diagrama o árbol de nodos del DOM. Se reconocen como nodos:

  1. El documento <html></html> (nodo raíz)

  2. Elementos HTML (etiquetas)

  3. Textos

  4. Comentarios

Accediendo a elementos del DOM

  1. Accediendo por ID del elemento

    const varID = document.getElementById("lista") console.log(varID) console.log(varID.innerHTML) console.log(varID.innerText) console.log(varID.tagName)

  2. Accediendo por clase del elemento

    const varClass = document.getElementsByClassName("elemento-lista") console.log(varClass) console.log(varClass[0]) console.log(varClass[0].id) console.log(varClass[0].className)

  3. Accediendo por elemento HTML

    const listTags = document.getElementsByTagName("ul") console.log(listTags)

  4. Accediendo por selector CSS

    const varSelector = document.querySelector(".code-block") console.log(varSelector)
    const varSelector = document.querySelector("#return-to-top") console.log(varSelector)
    const varSelector = document.querySelector("li p") console.log(varSelector)
