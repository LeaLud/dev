JavaScript

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

  • No re quiere ningún tipo de instalación, ya que la mayoría de los navegadores web lo incorporan
  • Archivo vinculoado ↗

Impresión de datos

  1. console.log("Hola Mundo")
    console.log(4)
    console.log(variable)
  2. Impresión de valores concatenados. Los "espacios" se deben agregar como strings
    console.log("Hola Mundo " + 4 + variable)

Variables

  1. Definiendo una variable estática
    const myVar = "Esta es una variable que no puede cambiar su valor"

  2. Definiendo una variable dinámica

    let myVar = "Esta es una variable que puede cambiar su valor"

Tipos de datos

  1. String. Cadena de caracteres

    let string = "Esto es una cadena de caracteres"
    let string = "Cadena de caracteres con \"comillas\""
    let string = 'Otra forma de indicar strings "comillas" dobles'

  2. Number. Números

    let string = 4

    1. Incrementando y reduciendo en 1 una variable numérica

      varNum++
      varNum--

    2. Incrementando y reduciendo una variable numérica y reasigándole el nuevo valor

      varNum+=5
      varNum-=5

  3. Boolean. Boleano, verdadero o falso

    let string = true | false

  4. Null. Valor nulo

    let string = null

  5. Undefined. Valor no definido. viene por defecto

    let string = undefined

  6. Object.

    let string =

  7. Symbol.

    let string =

Secuencias de escape

  1. \' Comilla simple
  2. \" Comilla doble
  3. \\ Barra invertida
  4. \n Salto de línea
  5. \r Retorno de carro
  6. \t Tabulación
  7. \b Retroceso
  8. \f Salto de página
