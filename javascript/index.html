JavaScript

Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Arrays y Arrays anidados

  1. Definir un array

    let myArray = ["Leandro", 37, 1986]
    let myArray2 = ["Federico", 9, 1986]
    let myArray3 = [["Leandro", 37, 1986], ["Federico, 9 , 1986"]]
    let myArray3 = [["Leandro", 37, 1986], myArray2]

    console.log(myArray3[1][2])

  2. Modificar el valor de un array

    let myArray = ["Leandro", 37, 1986]

    myArray[1] = 38

  3. Agregar un valor al comienzo de un array

    let myArray = ["Leandro", 37, 1986]

    myArray.unshift("Enero")

  4. Agregar un valor al final de un array

    let myArray = ["Leandro", 37, 1986]

    myArray.push("Enero")

  5. Quitar el primer valor de un array

    let myArray = ["Leandro", 37, 1986]

    myArray.shift()

  6. Quitar el último valor de un array

    let myArray = ["Leandro", 37, 1986]

    myArray.pop()
