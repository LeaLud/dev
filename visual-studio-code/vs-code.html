Extensiones útiles para Visual Studio Code

  • Live Server. Actualiza automáticamente en el navegador los desarrollos de VS Code
  • vscode-pdf. Muestra archivos PDF en VS Code
  • ESLint. Integra ESLint JavaScript en VS Code
  • ES7+ React/Redux/React-Native/JS snippets. Extensión de sintaxis para React, React-Native and Redux in JS/TS with ES7+

Paleta de comandos (F1) (Ctrl + Shift + P)

  • >Open Settings (UI). Abre el panel de configuración
  • >Open User Settings (JSON). Abre el archivo settings.json de configuración:
    { "editor.minimap.size": "fill", "editor.minimap.renderCharacters": false, "editor.minimap.enabled": false, "editor.renderWhitespace": "none", "diffEditor.hideUnchangedRegions.enabled": true, "window.menuBarVisibility": "compact", "liveServer.settings.donotShowInfoMsg": true, "liveServer.settings.donotVerifyTags": true, "editor.fontSize": 18, "files.autoSave": "afterDelay", "editor.stickyScroll.enabled": false, "editor.wordWrap": "on", "editor.detectIndentation": true, "workbench.tree.enableStickyScroll": true, "php.validate.executablePath": "C:/xampp/php/php.exe" }
  • >Format Document. Acomoda el código con espacios o tabulaciones
  • >Simple Browser: Show. Comando que abre un navegador en una pestaña de VS Code para visualizar en vivo los cambios que se hagan sobre la URL que se le indique
