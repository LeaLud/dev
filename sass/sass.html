SASS - Compilación de CSS

Compilar CSS transforma código escrito en preprocesadores (lenguajes más avanzados) en CSS estándar que los navegadores pueden entender. Las extensiones del CSS compilado pueden ser .SCSS o .SASS pero se recomienda .SCSS debido a la similitud de sintaxis con el .CSS normal

Programas necesarios

  1. Node.js para el desarrollo local

  2. Comprobar versión instalada en consola de comandos

    node -v
    node --version

  3. NPM (Node Package Manager). Instalar/actualizar la última versión

    npm install -g npm@latest

  4. Comprobar versión instalada

    npm -v
    npm --version

Instalación SASS

  1. npm install -g sass

    Comprobar versión instalada

    sass --version

Compilando las hojas de estilo

  1. Entrar por consola al directorio donde están los archivos .SCSS o .SASS

    cd C:/dev/sass/css

  2. Apuntar el archivo de entrada en busca de cambios y definir el archivo de salida .css

    sass --watch style.scss style.css

  3. Para deja de observar los cambios en el archivo

    Ctrl-C

Módulos

Son hojas de estilo separadas para distintos elementos. Por ejemplo, se puede definir un reset o el estilo de los banners por separado y luego integrarlos en la hoja principal style.scss

  1. _reset.scss

    _banners.scss

    @use 'reset'; @use 'banners';

Variables

  1. Definidas en _variables.scss

    // Colores $primary-color: #3498db; $secondary-color: #2ecc71; // Tamaños $width100: 100%; $width50: 50%; // Fuentes $font-primary: 'Helvetica Neue', Helvetica, Arial, sans-serif; $font-secondary: 'Georgia', serif;

  2. En cada hoja de estilo, llamar las variables

    _reset.scss

    _banners.scss

    @use 'variables' as *;
