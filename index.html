Elementos HTML5

  1. <main></main> indica el contenido principal de la página
  2. <article></article> contenido independiente, noticias, publicaciones de blogs
  3. <section></section> agrupa contenido relacionado dentro de un documento
  4. <aside></aside> agrupa contenido secundario en barras laterales
  5. <small></small> fuente de letra reducida. Puede usarse para el Copyright
  6. <img src="img/paisaje2.jpg" alt="Paisaje" loading="lazy/eager"/> fuente de letra reducida. Puede usarse para el Copyright
  7. <audio src="resources/bonus-super-mario-3.mp3" controls autoplay muted loop style="width: 50%; margin: 20px 0"></audio> indica medios de audio
  8. <video src="resources/emi-control-diabetes.mp4" poster="img/paisaje2.jpg" controls autoplay muted loop width="50%"></video> indica medios de video
  9. <picture></picture> para indicar medios de imagen

Etiquetas escenciales del <HEAD></HEAD>

<head>
<!--Metadata-->
<meta charset="utf-8">
<meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
<meta name="theme-color" content="#000">

<!--Seo-->
<meta name="author" content="Leandro Ludueña">
<meta name="description" content="Página resumen de Bootstrap">
<meta name="keywords" content="Bootstrap, implementación de Bootstrap, usos de Bootstrap">
<link rel="alternate" href="https://mipagina.com/en" hreflang="en-EN">
<link rel="canonical" href="https://mipagina.com/">
<meta name="robots" content="index, follow">
<title>Etiquetas del head del HTML</title>

<!--Open Graph-->
<meta property="og:title" content="Etiquetas del head del HTML">
<meta property="og:description" content="Página resumen de Bootstrap">
<meta property="og:image" content="/img/image.jpg">
<meta property="og:image:alt" content="Logo Bootstrap">

<!--Css-->
<link rel="stylesheet" type="text/css" href="../style/style.css" media="(max-width:991px)">
<link rel="stylesheet" type="text/css" href="../style/style.css" media="(min-width:992px)">
<link rel="stylesheet" type="text/css" href="style/scss/custom.css" media="all">

<!--Favicon-->
<link rel="shortcut icon" type="image/x-icon" href="favicon.png">
</head>
