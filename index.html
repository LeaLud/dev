Elementos HTML5

  1. <main></main> indica el contenido principal de la página
  2. <article></article> contenido independiente, noticias, publicaciones$border-width de blogs
  3. <section></section> agrupa contenido relacionado dentro de un documento
  4. <aside></aside> agrupa contenido secundario en barras laterales
  5. <audio></audio> para indicar medios de audio
  6. <video></video> para indicar medios de video
  7. <picture></picture> para indicar medios de imagen
