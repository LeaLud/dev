Bootstrap

  • Librería de recursos para el diseño y maquetado front-end de sitios web

Instalación desde CDN via jsDelivr

  • Incluir en el <head></head> las hojas de estilo y JavaScript. La librería JavaScript puede ser colocada justo antes de la etiqueta </body> para evitar conflictos
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-T3c6CoIi6uLrA9TneNEoa7RxnatzjcDSCmG1MXxSR1GAsXEV/Dwwykc2MPK8M2HN" crossorigin="anonymous" />

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/js/bootstrap.bundle.min.js" integrity="sha384-C6RzsynM9kWDrMNeT87bh95OGNyZPhcTNXj1NW7RuBCsyN/o0jlpcV8Qyq46cDfL" crossorigin="anonymous"></script>

Instalación desde archivos fuente

  • Descargar los archivos y llamarlos desde el <head></head>

Componentes e iconos

  • Componentes: carrusel, botones dropdown, menú acordeón, etc.
  • Insertar iconos desde CDN en el <head></head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css" />
  • Importar iconos desde CDN en la hoja de estilos
    @import url("https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.1/font/bootstrap-icons.css");
  • Instalar el paquete de iconos desde el gestor de paquetes con npm o Composer
    npm i bootstrap-icons
    composer require twbs/bootstrap-icons

Instalación de SASS

  • Recurso que permite a CSS usar variables y funciones
  • Requiere Node.js para la instalación via npm por consola
  • npm install -g sass
  • Crear un archivo .SCSS (recomendado) o .SASS e importarle los .SCSS de Bootstrap necesarios. Se recomienda importar el general completo para incluir todas las funcionalidades
    @import "bootstrap";

Compilando hoja de estilos de SASS

  • Entrar por consola en la carpeta del archivo a compilar y ejecutar el siguiente comando
    sass --watch ARCHIVOENTRADA.EXTENSION ARCHIVOSALIDA.CSS
  • Una vez exportado el archivo, Sass vigila los futuros cambios para aplicarlos automáticamente al .CSS de salida
  • Para dejar de monitorear el archivo de entrada, en consola pulsar CTRL+C
  • No modificar los estilos originales de Bootstrap, solo los .SCSS con funciones

Anidación de selectores en una de estilo SCSS

/*anidación de selectores*/
main{
p{
font-size:auto;
}
a{
font-size:auto;
} li{
font-size:auto;
}
}
