Consola de comandos (Win + R + CMD)

  • cd muestra la ruta del directorio actual
  • cd DIRECTORIO cambia de directorio
  • cd.. sube un nivel de directorio
  • tree lista la estructura de directorios de toda la instalación de Windows
  • dir lista la estructura de directorios de la ubicación actual
  • copy con ARCHIVO.EXTENSION crea un archivo y se procede a introducir su contenido en texto. Se genera el archivo al escribir al menos 2 líneas y presionar luego Ctrl+C
  • md CARPETA crea una carpeta
  • mkdir CARPETA crea una carpeta
  • rename ARCHIVO.EXTENSION NUEVOARCHIVO.EXTENSION | CARPETA NUEVACARPETA cambia el nombre al archivo o carpeta
  • move ARCHIVO.EXTENSION | CARPETA RUTA | CARPETA mueve un archivo o carpeta a una ruta o carpeta
  • del ARCHIVO.EXTENSION borra un archivo
  • erase ARCHIVO.EXTENSION borra un archivo
  • rmdir CARPETA borra una carpeta
  • chkdsk UNIDAD chequea el estado de una unidad y reparar errores
  • code . abre el editor de código dentro de la carpeta posicionada
  • PROGRAMA --version muestra la versión del programa indicado si es que existe una instalación
  • ipconfig muestra la configuración IP de Windows
  • format UNIDAD formatea una unidad
  • format UNIDAD /Q formatea rápido una unidad
  • Ctrl + C termina una operación por lotes
  • cls limpia la consola
  • exit cierra la consola
